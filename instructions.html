📖 A BUSCA PELO LEITÃO

O Leitão foi capturado e está perdido em algum lugar deste vasto mapa! Seu objetivo como Pooh é coletar todo o mel espalhado pelo caminho. Com mel suficiente (5 potes), uma chave aparecerá e você poderá libertar seu melhor amigo antes que o relógio zere.

🎮 COMO SE MOVER

⬅️➡️ Use as Setas (ou A e D) para andar.
⬆️ Use a Seta Cima (ou Espaço) para Pular.
↩️ Aperte ENTER para avançar em diálogos.

🐝 PERIGOS E PRÊMIOS

🍯 Mel: Colete 5 para invocar a chave.
🔑 Chave: Pegue para avançar de fase!
🐝 Abelhas: Encostou, perdeu vida.
🗡️ Espinhos: Morte instantânea!

💡 SEGREDOS DA FLORESTA

  • ⏱ Atalhos (Game Over): Pressione 'R' para tentar a fase atual novamente, ou 'S' para reiniciar todo o jogo do zero. Para voltar para o menu rápido, aperte 'H'.
  • 🎬 Histórias: Pressione 'P' a qualquer momento durante os diálogos de história para pular a cutscene e ir direto para as fases.
  • 🛡️ Sobrevivência: Se você for ferido por uma abelha, o Pooh piscará e ficará invulnerável por exato 1 segundo. Use isso para se reposicionar em segurança!
  • ⏳ Ritmo do Jogo: O tempo total para encontrar todas as chaves é de duraduros 185 segundos. Acelere nas fases de plataformas móveis!
