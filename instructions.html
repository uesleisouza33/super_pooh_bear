📖 A BUSCA PELO LEITÃO (CO-OP)

O Leitão foi capturado e está perdido no mapa! O jogo agora conta com Modo Cooperativo Local. O P1 (Pooh) deve coletar todo o mel espalhado pelo caminho. O P2 (Tigrão) deve ajudar a liberar o percurso. Com 5 potes, uma chave aparecerá e vocês poderão libertar seu amigo!

🎮 COMO SE MOVER

🐻 P1 (POOH)

A D Andar para os lados.
W Pular (ou Espaço).

🐯 P2 (TIGRÃO)

⬅️➡️ Andar para os lados.
⬆️ Pular pelas plataformas.
↩️ ENTER avança as falas.

🐝 PERIGOS E PRÊMIOS

🍯 Mel: Colete 5 para invocar a chave.
🔑 Chave: Pegue para avançar de fase!
🐝 Abelhas: Encostou, perdeu vida.
🗡️ Espinhos: Morte instantânea!

💡 SEGREDOS DA FLORESTA

  • 👯 Equipe: A Câmera trava se Pooh e Tigrão se separarem muito! Andem sempre juntos e usem o Tigrão de degrau.
  • ⏱ Atalhos: Pressione 'R' para tentar a fase novamente, ou 'S' para reiniciar tudo do zero. Para menu rápido aperte 'H'.
  • 🎬 Histórias: Pressione 'P' durante os diálogos para pular a cutscene e ir direto para a fase.
  • 🛡️ Sobrevivência: Se você for ferido por uma abelha, o Pooh piscará e ficará invulnerável por exato 1 segundo. Use isso para se reposicionar!
